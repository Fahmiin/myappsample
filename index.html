This is my app!

  1. Use 'touch' to make new files
  2. Use 'git init' to initialize git
  3. Add user name and user email through 'git config --global user.name/email Name/email'
  4. Adding new files to git, use 'git add (name of file)'
  5. 'git status' to check the current status of git
  6. Remove file from staging area 'git rm --cached name of file'
  7. 'git add*.hmtml' to ad all html files to staging area and works for all other types of files
  8. 'git add .' to add all files to staging area
