  1. Use 'touch' to make new files
  2. Use 'git init' to initialize git
  3. Add user name and user email through 'git config --global user.name/email Name/email'
  4. Adding new files to git, use 'git add (name of file)'
  5. 'git status' to check the current status of git
  6. Remove file from staging area 'git rm --cached name of file'
  7. 'git add*.hmtml' to ad all html files to staging area and works for all other types of files
  8. 'git add .' to add all files to staging area
  9. To commit:
    • 'git commit'
    • press i in vim file to add comment
    • esc to go out of insert mode
    • :wq to commit
  10. To commit fast:
    • use 'git commit' -m Comment
  11. Add a '.gitignore' file and add different things into the file to ignore adding to staging area
  12. Branches:
    • 'git branch name' to create new branch
    • 'git checkout name' to move to new branch
    • 'git merge name' to merge to master branch
    • Add a '.gitignore' file to ignore any files added without adding to staging area
    • Branching:
      • 'git branch name' to create a new branch
      • 'git checkout name' to switch onto the branch
    • Merging, use 'git merge name'
    • Creating remote repository on github
      • Check out the steps
      • Use 'git push' to add new commits to github
    • Use the 'git pull' to check for updates
